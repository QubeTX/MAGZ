     Plan: On-Brand 404 Page + Comprehensive Design System Reference

     Context

     The MAGZ website (React 19 + Vite 7 + Tailwind CSS 4, brutalist sports
     marketing aesthetic) has a minimal placeholder 404 page — just a centered
     box with "404" and "Page Not Found". Every other page in the site (Home,
     CareersPage) has rich Framer Motion animations, Navbar/Footer, and full
     branded design language. The 404 page needs to match.

     Additionally, the project lacks a centralized design system reference. As
     new pages get added, there's no single document to ensure consistency. A
     DESIGN.md file will serve as the canonical reference for brand identity,
     animation patterns, typography, layout conventions, and component
     architecture.

     1.7 Animation System (Framer Motion)

     Standard easing: [0.76, 0, 0.24, 1] — used consistently across all
     components

     Reusable variants:
     // Stagger container (Services, CareersPage)
     const stagger = { hidden: {}, show: { transition: { staggerChildren:
     0.1-0.15 } } };

     // Fade up (universal entrance)
     const fadeUp = { hidden: { opacity: 0, y: 30-40 }, show: { opacity: 1, y:
     0, transition: { duration: 0.6, ease: [0.76, 0, 0.24, 1] } } };

     // Slide up (Services variant)
     const slideUp = { hidden: { opacity: 0, y: 40 }, show: { opacity: 1, y: 0,
     transition: { duration: 0.6, ease: [0.76, 0, 0.24, 1] } } };

     // Scale reveal (Brands grid)
     const itemVariants = { hidden: { opacity: 0, scale: 0.8 }, show: { opacity:
      1, scale: 1, transition: { duration: 0.5, ease: [0.76, 0, 0.24, 1] } } };

     Entrance patterns:
     - Navbar: initial={{ y: -100 }} animate={{ y: 0 }} duration 0.8
     - Hero text: initial={{ y: 80, opacity: 0, scale: 0.9 }} duration 1
     - Letter spacing reveal: initial={{ letterSpacing: "0.8em", opacity: 0 }} 
     animate={{ letterSpacing: "0.3em", opacity: 1 }} duration 1.2
     - Section headings: initial={{ opacity: 0, y: 30 }} whileInView={{ opacity:
      1, y: 0 }} viewport once
     - Side reveals: initial={{ opacity: 0, x: -50 }} or x: 50 for left/right
     - Staggered items: delay 0.3 + i * 0.1 (nav), i * 0.08 (Gallery columns), i
      * 0.12 (Stats)
     - Mobile menu: clipPath circle animation from circle(0% at calc(100% - 
     40px) 40px) to circle(150%...)

     Scroll-based:
     - Parallax: useScroll + useTransform(scrollYProgress, [0,1], ["0%","30%"])
     for background, ["0%","50%"] for text
     - Opacity fade: useTransform(scrollYProgress, [0, 0.8], [1, 0])
     - Gradient line scale: initial={{ scaleX: 0 }} whileInView={{ scaleX: 1 }}
     - Gradient bar width: initial={{ width: 0 }} whileInView={{ width: 96-120 
     }}

     Interactive:
     - Hover scale: whileHover={{ scale: 1.02-1.1 }}
     - Tap scale: whileTap={{ scale: 0.95-0.98 }}
     - Hover translate: whileHover={{ x: 4-8 }} (menu items, cards)
     - Spring physics: type: "spring", stiffness: 300-400 (interactive text,
     stats)
     - Hover color: whileHover={{ color: "#F7941D" }} (menu), color: "#2D6BC6"
     (socials)

     Accordion expand/collapse:
     initial={{ height: 0, opacity: 0 }}
     animate={{ height: "auto", opacity: 1 }}
     exit={{ height: 0, opacity: 0 }}
     transition={{ duration: 0.4-0.5, ease: [0.76, 0, 0.24, 1] }}

     Infinite animations:
     - Rotating squares: animate={{ rotate: [0, 90, 0] }} duration 4, easeInOut
     - Counter-rotating: rotate: [0, -90, 0] with delay 0.5
     - Pulsing square: animate={{ scale: [1, 1.3, 1] }} duration 2
     - Scroll indicator bounce: animate={{ y: [0, 6, 0] }} duration 1.5
     - Marquee scroll: animate={{ x: ["0%", "-50%"] }} linear, duration 15-20
     - Icon rotation on toggle: animate={{ rotate: isOpen ? 180 : 0 }}

     1.8 Smooth Scroll (Lenis)

     - Initialized in App.tsx with duration: 1.2, exponential easing
     - Exposed as window.__lenis for programmatic scrolling
     - Components access via (window as any).__lenis
     - lenis.scrollTo(element, { offset: -80 }) for nav links (clears navbar)
     - lenis.scrollTo(element, { offset: -100, duration: 1 }) for accordions
     - lenis.scrollTo(0) for scroll-to-top
     - Accordion scroll must use setTimeout(fn, 500) to wait for expand
     animation

     1.9 Pretext Heading Measurement

     - @chenglou/pretext dynamically imported for responsive text measurement
     - Used in PretextHeading.tsx (Hero "MAGZ") and Hero subtitle line-break
     decision
     - Pattern: pretext.prepare(text, fontString) → pretext.layout(prepared, 
     containerWidth, lineHeight) → use result.height to determine layout
     - Requires optimizeDeps.include in Vite config and
     allowImportingTsExtensions: true in tsconfig
